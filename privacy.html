Privacy Policy

WiredWrong Publisher — Last updated: March 2026

1. Overview

WiredWrong Publisher is a private, single-user tool. It does not collect, process, or share any personal data from TikTok users or any other individuals.

2. Data We Collect

CookiesNoThis is a CLI tool — no browser or cookies

3. TikTok API Scopes Used

  • video.upload — to send video files to TikTok servers
  • video.publish — to publish the creator's own videos
  • user.info.basic — included by Login Kit to confirm creator identity

These scopes are used only to post original content to the creator's own account (@wiredwrong_b15). No other data is accessed or retained.

4. Token Storage

OAuth access and refresh tokens are stored in a local configuration file on the creator's private machine. They are never transmitted to any server other than TikTok's official API endpoints.

5. Third Parties

No data is shared with any third parties. The only external API calls made are to TikTok's official Content Posting API.

6. Children's Privacy

This tool is not intended for use by anyone other than the creator. No data from children is collected.
